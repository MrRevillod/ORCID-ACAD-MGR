# Requerimientos Funcionales

## RF-01 - Registrar Investigador

El sistema deberá permitir registrar un investigador utilizando su identificador ORCID.

### Flujo

1. El administrador ingresa el ORCID ID.
2. El sistema consulta ORCID.
3. El sistema obtiene la información pública disponible.
4. El sistema crea el registro del investigador.
5. El sistema importa sus publicaciones asociadas.

El sistema deberá permitir administrar las siguientes entidades:
